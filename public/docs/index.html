Introduction

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

Register a new user

Registers a new user and sends an email for account pending approval.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/v3/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"user_name\": \"johndoe\",
    \"email\": \"johndoe@example.com\",
    \"ecclesia_id\": 17,
    \"first_name\": \"John\",
    \"last_name\": \"Doe\",
    \"middle_name\": \"A.\",
    \"address\": \"123 Main St\",
    \"phone_number\": \"1234567890\",
    \"city\": \"Springfield\",
    \"state\": \"Illinois\",
    \"address2\": \"Apt 4B\",
    \"country\": \"USA\",
    \"zip\": \"62704\",
    \"email_confirmation\": \"johndoe@example.com\",
    \"password\": \"Password@123\",
    \"password_confirmation\": \"Password@123\"
}"

const url = new URL(
    "http://127.0.0.1:8000/api/v3/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "user_name": "johndoe",
    "email": "johndoe@example.com",
    "ecclesia_id": 17,
    "first_name": "John",
    "last_name": "Doe",
    "middle_name": "A.",
    "address": "123 Main St",
    "phone_number": "1234567890",
    "city": "Springfield",
    "state": "Illinois",
    "address2": "Apt 4B",
    "country": "USA",
    "zip": "62704",
    "email_confirmation": "johndoe@example.com",
    "password": "Password@123",
    "password_confirmation": "Password@123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),

Body Parameters

user_name   string   

Unique username for the user. Example: johndoe

email   string   

Unique email address. Example: johndoe@example.com

ecclesia_id   integer  optional  

nullable Ecclesia ID if applicable. Example: 17

first_name   string   

First name of the user. Example: John

last_name   string   

Last name of the user. Example: Doe

middle_name   string  optional  

nullable Middle name of the user. Example: A.

address   string   

Address of the user. Example: 123 Main St

phone_number   string   

User's phone number. Example: 1234567890

city   string   

City of residence. Example: Springfield

state   string   

State of residence. Example: Illinois

address2   string  optional  

nullable Additional address information. Example: Apt 4B

country   string   

Country of residence. Example: USA

zip   string   

Zip code. Example: 62704

email_confirmation   string   

Confirmation of the email address. Must match email. Example: johndoe@example.com

password   string   

Password for the user. Must be at least 8 characters and include one special character (@$%&). Example: Password@123

password_confirmation   string   

Confirmation of the password. Must match password. Example: Password@123
